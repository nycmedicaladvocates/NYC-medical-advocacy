OVERDOSE PREVENTION

Narcan (naloxone) — basics

Narcan can reverse opioid overdoses. If you think someone is overdosing, call 911 immediately.

Request help finding Narcan Back to Substance Use

What to do

Overdose response steps
  1. Call 911
  2. Give naloxone (Narcan) if available
  3. Stay with the person until help arrives

If you are unsure, call 911 — it’s better to be safe.
